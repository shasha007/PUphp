#
# ThinkSNS - 迎新 安装说明
#

1. 上传 newcomer 文件夹至/apps/目录
2. 进入"管理后台 - 应用 - 应用管理 - 应用安装", 点击"安装", 应用状态：默认
3. 如果不是第一次安装, 请进入"管理后台 - 扩展", 点击"缓存更新"
4. 进入"管理后台 - 全局 - 注册配置 - 注册方式" 请使用"公开注册"方式
5. 进入"管理后台 - 应用 - 应用配置 - 迎新", 默认的8篇文章, 分别对应迎新首页8个按钮, 请勿删除, 修改为相应内容

使用
1.应用状态最好置为默认, 如果是可选, 用户未安装迎新应用, 且登录的状态下无法查看迎新的资讯
2.安装后请在后台修改资讯, 或者添加新资讯, 请勿删除默认的8个
3.用户注册完成后, 学校,院系和学号在个人空间的详细资料中可以查看, 给新生增加了一些标签,  标签内容为"2012新生",学校名, 院系名, 可以按标签搜索

access.inc.php中增加
'document/Index/index'		=> true, // 文库首页
'appstore/Index/index'		=> true, // 应用首页		
'newcomer/Index/*'		=> true, // 迎新首页
