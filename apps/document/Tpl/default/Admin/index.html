开启Word,PPT在线阅读：

需要在服务器环境配置Word,PPT转换功能后再开启。否则可能会影响系统正常

	
	安装说明
	1.修改分类有可能需要刷新缓存才能生效, 请进入"管理后台 - 扩展", 点击"缓存更新"
	2.应用状态请置为默认
	3.请将临时文件目录权限设为777
	4.重新安装后, 请刷新缓存, 并清空掉临时文件目录
	
	
	环境配置
	1. 支持pdf在线阅读功能
	安装 SwfTools
	最新版本为 0.9.2
	下载地址: http://www.swftools.org/download.html
	2. word,ppt转换功能
	安装 Java 环境 和 OpenOffice, 
	Java下载地址: http://www.java.com/
	OpenOffice最新版本为 3.4.1
	下载地址: http://www.openoffice.org/download/
	3. 将上述应用路径写入对应平台的配置文件: Lib/Action/php/config/config.ini.*.php
	linux环境下为config.ini.nix.php
